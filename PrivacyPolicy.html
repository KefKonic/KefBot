Privacy Policy for KefBot

Last Updated: 2025/03/20

1. Introduction

Welcome to KefBot! This Privacy Policy explains how KefBot collects, uses, and protects user data while in operation on Discord servers. By using KefBot, you agree to the terms outlined in this policy. If you do not agree, please discontinue using the bot.

2. Data Collection

KefBot collects and processes the following types of data:

  • User Information: Usernames, user IDs
  • Server Information: Server IDs, server settings
  • Messages: Messages sent in channels where logging is enabled

This data is collected automatically as part of KefBot’s logging and moderation functionalities.

3. How Data is Used

Collected data is used for the following purposes:

  • Providing bot functionality, including moderation and server logs
  • Storing and applying server preferences
  • Preventing abuse and ensuring security within Discord communities
  • Enhancing bot performance and user experience

KefBot does not share, sell, or distribute user data to third parties. The data collected is strictly used for operational purposes.

4. Data Storage & Security

  • Logged messages, IDs, and names are stored securely on the bot’s hosting environment.
  • Data retention periods vary based on server settings, but logs are not stored indefinitely and may be periodically purged.
  • Reasonable security measures are in place to prevent unauthorized access to stored data.

5. User Rights & Data Removal

Users or server administrators may request data removal by contacting the bot developer. Requests should include relevant details (e.g., user ID, server ID) to facilitate the removal process.

If a server administrator disables logging or removes the bot, all collected data related to that server will be deleted within a reasonable timeframe.

6. Third-Party Services

KefBot interacts with the Discord API, which means some data (such as messages and user IDs) is processed by Discord. For more information, please refer to Discord’s Privacy Policy.

No additional third-party services process or store data from KefBot beyond what is necessary for bot operation.

7. Policy Updates

This Privacy Policy may be updated periodically. Users will not be individually notified of changes, so it is recommended to review this policy regularly. Continued use of KefBot after updates constitutes acceptance of the revised policy.

8. Contact Information

For questions, concerns, or data removal requests, please contact the bot developer at KefKonic@Gmail.com.

By using KefBot, you acknowledge that you have read, understood, and agreed to this Privacy Policy.
